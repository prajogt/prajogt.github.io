Project Details

Thrive: Happy Habits

Developed with Robbie Lowenstein @ Bellwoods Strategic Capital. A React Native application using Expo to be on iOS and Android app stores, with a Django API server containerized in Docker and hosted on Heroku. Utilized a PostgreSQL database.

Order-Up

An order management system complete with results/statistics page. Made in Java using Spring Boot for the backend and JavaFX for the application frontend. Demonstrating OOP and clean architecture practices.

Branch-Out

A web based contacts page in the style of a family tree, where nodes reference contact info for other Users/family members. Built on the Next JS framework with React. Data stored using MongoDB.

Bed'n Breakfast

An AirBnB replica. Focus on data management with MySQL and embedded SQL in Java. Additional features include linear regression for price suggestions and Apache OpenNLP for natural language processing of user comments.

ML Course Project

A machine learning project, employing KNN, probabilistic models, and neural networks. Implemented, modified, and analyzed models in Python using PyTorch, Sci-Kit, and numpy using data provided by Eedi, an online education platform.
